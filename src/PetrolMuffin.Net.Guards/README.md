# PetrolMuffin.Net.Guards

Library to provide a simple way to check and throw exceptions for different scenarios.

## Installation

```bash
dotnet add package PetrolMuffin.Net.Guards
```

## Features

- **ThrowIf** — guard methods for arguments, streams, and variables
- **ThrowHelper** — helper for throwing exceptions

## Dependencies

- PetrolMuffin.Net.Extensions
- PetrolMuffin.Net.Exceptions
