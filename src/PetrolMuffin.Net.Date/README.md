# PetrolMuffin.Net.Date

Common types and interfaces for working with dates and periods in .NET.

## Installation

```bash
dotnet add package PetrolMuffin.Net.Date
```

## Features

- **Period** — date period mode
- **PeriodOffset** — period with offset support
- **PeriodExtensions** / **PeriodOffsetExtensions** — extension methods for periods
- **DateMath** — date calculations

## Dependencies

- PetrolMuffin.Net.Extensions
- PetrolMuffin.Net.Guards
- PetrolMuffin.Net.Helpers
