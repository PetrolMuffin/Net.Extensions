# PetrolMuffin.Net.Scaffolding

Base types and utilities for building .NET applications.

## Installation

```bash
dotnet add package PetrolMuffin.Net.Scaffolding
```

## Features

- **StateHolder** — holder for state management
- **Disposable** — base disposable implementation
- **AsyncDisposable** — base async disposable implementation
