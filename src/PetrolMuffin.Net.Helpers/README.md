# PetrolMuffin.Net.Helpers

Library with helper functions for working with different types of data.

## Installation

```bash
dotnet add package PetrolMuffin.Net.Helpers
```

## Features

- **EmbeddedResourceReader** — static helper for reading embedded resources

## Dependencies

- PetrolMuffin.Net.Guards
