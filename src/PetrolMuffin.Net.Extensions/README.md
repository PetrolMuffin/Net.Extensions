# PetrolMuffin.Net.Extensions

Extension methods for different types.

## Installation

```bash
dotnet add package PetrolMuffin.Net.Extensions
```

## Features

- **EnumExtensions** — extension methods for enumerations
- **CollectionExtensions** — extension methods for collections
