# PetrolMuffin.Net.Threading.Events

Multithreading for .NET events.

## Installation

```bash
dotnet add package PetrolMuffin.Net.Threading.Events
```

## Features

- **AsyncEventHandler** — async event handler delegate
- **AsyncEventHandlerExtensions** — extension methods for async event handling

## Dependencies

- PetrolMuffin.Net.Extensions
- PetrolMuffin.Net.Threading
