# PetrolMuffin.Net.Threading

Multithreading for .NET.

## Installation

```bash
dotnet add package PetrolMuffin.Net.Threading
```

## Features

- **TaskExtensions** — extension methods for `Task` and async operations
- **ReaderWriterLockSlimExtensions** — extension methods for `ReaderWriterLockSlim`

## Dependencies

- PetrolMuffin.Net.Extensions
