# PetrolMuffin.Net.Exceptions

Library with custom exception classes.

## Installation

```bash
dotnet add package PetrolMuffin.Net.Exceptions
```

## Features

- **NotInitializedException** — thrown when an object is used before initialization
- **CollectionNullOrEmptyException** — thrown when a collection is null or empty
- **StringNullOrEmptyException** — thrown when a string is null or empty
